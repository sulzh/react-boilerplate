# React Boilerplate

###### Usage

1. Clone repository

```
git clone --depth=1 <repository-name> <folder-name>
```

2. Remove git

```
rm -rf .git
```

3. Init git

```
git init --initial-branch=<branch-name>
git add .
git commit -m "Initial commit"
```

4. Connect repo to github

5. Install dependencies

```
npm i
```
